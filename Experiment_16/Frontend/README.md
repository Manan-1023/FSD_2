# Experiment 16: Testing Frontend and Backend Sections

## GOAL

To execute testing on both the frontend and backend sections of a web application employing current testing frameworks:

- Testing backend APIs with **pytest**
- Testing frontend forms with **Vitest + React Testing Library**

## TOOLS AND FRAMEWORKS

- React.js (Vite)
- Vitest
- @testing-library/react
- @testing-library/jest-dom
- Material UI (MUI)

### Development Utilities

- VS Code
- Terminal / CLI
